### 1. 主打主题：冷雾寂静风（最常见、最贵的那种）
最适合极简、生活类、阅读、效率、冥想、轻奢电商 App
（The Row / Loro Piana / 小红书顶级博主都在用的感觉）

| 用途              | 色号              | 颜色名称                | 备注                      |
| ----------------- | ----------------- | ----------------------- | ------------------------- |
| 主要背景          | #F5F5F5 → #D8D7D4 | 月灰白/极光白           | 几乎白，但比纯白更有温度  |
| 次级背景/卡片     | #D0D8E0 → #C3D7E8 | 冰川薄蓝                | 层次最明显的一层          |
| 主体文字/标题     | #6E7F85 → #8B9A9F | 风暴灰蓝                | 深但不黑，显高级          |
| 主要品牌色/强调色 | #98AFC7           | 雾霾灰蓝（C位主打色！） | 按钮、图标、选中状态必用  |
| 辅助强调色        | #A7B4C9           | 烟雨蓝                  | 副按钮、链接、tag         |
| 浅色辅助文字      | #B0C4DE           | 轻钢蓝                  | placeholder、次要说明文字 |

效果：冷冽、干净、极安静，一眼就是“贵”。

### 2. 深色模式主题（夜间最贵的那种）
直接把亮色模式颜色整体压暗+降饱和度：

| 用途          | 色号    | 备注             |
| ------------- | ------- | ---------------- |
| 页面背景      | #1C2A33 | 接近黑但带蓝调   |
| 卡片背景      | #2D3F47 | 比背景略提亮     |
| 次级卡片      | #3D5059 |
| 主要文字      | #C3D7E8 | 冰川薄蓝当白字用 |
| 标题/强调文字 | #98AFC7 | 主打雾霾灰蓝     |
| 选中/品牌色   | #A7B4C9 | 烟雨蓝高亮       |

效果：夜间模式里最不刺眼、最沉静的那种深色。

- Light Mode → 上面第 1 套冷雾寂静风
- Dark Mode → 上面第 2 套深色模式
